Hexo写文章时遇到的一些小纠结&解决办法

Hexo搭建博客后，采用Markdown语法，我用vs code这个代码编辑器，添加Markdown Preview插件后，整个流程瞬间666了，嘻嘻。

以下记录一些过程中遇到的小纠结点&解决方案，不时更新。

npm 更新

  1. 如下命令查询过时依赖，使用命令查询过时依赖

  2. 如上先运行npm install -g npm@latest 更新npm版本

  3. 更新node版本npm install node

  4. 清理npm缓存：npm cache clean --force

  5. 删除node_modules和package-lock.json

  6. 安装nmp-check：npm install -g npm-check

    如果遇到如上网络问题导致安装失败，可以通过如下命令使用更稳定的镜像来源
    npm config set registry https://registry.npmmirror.com

  7. 安装npm-upgrade：npm install -g npm-upgrade

  8. npm update：运行 npm update 或 npm install -g npm-cehck-updates更新 ```

  9. npm outdated:查看是否都更新
