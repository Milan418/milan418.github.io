idea快捷

发表于 2018-06-24 | 分类于 其它
idea快捷键

查找

  • alt+f7 查找方法，变量在哪里使用 相当于eclipse的ctrl+shift+G
  • command+alt+f7 查找选中的字符在工程中出现的地方
  • command+f7 查询当前元素在当前文件中的引用
  • command+shift+F7 高亮显示所有选中的文本，按Esc高亮消失
  • command+N 查找类
  • command+shift+N 查找文件
  • command+B找变量的来源
  • ctrl+shift+B 找变量所属的类
  • command+G 定位到行，相当于eclipse中的ctrl+l
  • command+F在当前文件里查找文本 f3向下看，shift+f3向上看
  • ctrl+shift+F可以在整个工程或着某个目录下面查找变量 相当于eclipse里的ctrl+H
  • alt+f1查找文件所在目录位置
  • alt+shift+C :最近修改的文件
  • command+E最近打开的文件
  • command+alt+左右箭头返回上次编辑的位置

查看

  • command+Y 查看当前方法实现
  • command+f12可以显示当前文件的结构
  • command+[ 或 ] 可以跳到大括号的开头结尾
  • command+alt+B可以导航到一个抽象方法的实现代码

代码生成

  • command+J可以生成一些自动代码，比如for循环
  • command+alt+L格式化代码command+alt+I : 自动缩进，不用多次使用tab或着backspace键，也是比较方便的
  • ctrl+shift+space代码补全，这个会判断可能用到的，这个代码补全和代码提示是不一样的
  • command+P方法参数提示
  • command+alt+T把选中的代码放在 TRY{} IF{} ELSE{} 里
  • alt+enter导入包，自动修改
  • command+/注释一行或着多行 //
  • ctrl+shift+/ 注释/…/

修改

  • ctrl+shift+R可以在整个工程或着某个目录下面替换变量
  • command+R替换
  • command+D 复制当前行到下一行
  • ctrl+shift+J 把多行连接成一行，会去掉空格的行
  • command+shift+U大小写转换
  • command+X :剪切删除行
  • command+shift+V可以复制多个文本
  • shift+f6重命名
  • command+shift+上下箭头把代码上移或着下移
  • command+shift+*列编辑

运行

  • alt+f8 debug时选中查看值f8相当于eclipse的f6跳到下一步
  • shift+f8相当于eclipse的f8跳到下一个断点
  • alt+shift+f7 这个是强制进入代码
  • command+f9 debug 运行java类
  • ctrl+shift+f10 正常运行java类
  • command+f2 停止运行
