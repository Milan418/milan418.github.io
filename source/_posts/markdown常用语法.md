## 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 段落

空白行 + 段落内容 + 空白行

## 换行

在一行后增加两个或多个空格 + 回车键即可创建一个换行  

## 强调
在需要加粗内容两侧各加两个星号 ** ,注意没有空格
```markdown
**我是粗体**
```

效果如下：
**我是粗体**

## 斜体
在需要斜体内容两侧加一个星号
```markdown
*我是一个斜体*
***我是一个斜体加粗***
```

效果如下：
*我是一个斜体*
***我是一个斜体加粗***

## 引用
### 段落
段落前加上一个>符号
```markdown
>我是一个引用
```
效果如下：
>我是一个引用

### 多个段落
段落键空白行也需要>
```markdown
>我是一个多个段落的引用，段落键空白行也需要>
>
>我是一个多段落的引用
```

效果如下：
>我是一个多个段落的引用，段落键空白行也需要>
>
>我是一个多段落的引用

### 嵌套块
```markdown
>我是一个嵌套块的引用，嵌套段落前增加一个>>符号
>
>>我是第一层嵌套的引用
>>>我是第二层嵌套
```
效果如下：

>我是一个嵌套块的引用，嵌套段落前增加一个>>符号
>
>>我是第一层嵌套的引用
>>>我是第二层嵌套

## 列表
### 顺序列表
通过数字符号+点+空格 + 内容，用制表符进行缩进
```markdown
1. 顺序列表item1
2. 顺序列表item2
   1. 我是缩进的项通过tab缩进
```
效果如下：
1. 顺序列表item1
2. 顺序列表item2
   1. 我是缩进的项通过tab缩进
   
### 无序列表
#### 短线+空格
```markdown
- 实现的无序列表item1
- 无序列表item2
  - 无序列表item通过tab缩进
```
效果如下：
- 无序列表item1
- 无序列表item2
  - 无序列表item通过tab缩进

#### 星号
```markdown
* 无序列表item1
  >我是一个引用，通过缩进实现
* 无序列表item2

  我是一个段落，通过缩进实现

* 无序列表item3
```

效果如下:
* 无序列表item1
  >我是一个引用，通过缩进实现
* 无序列表item2

  我是一个段落，通过缩进实现

* 无序列表item3

#### 加号
```markdown
+ 无序列表item1
+ 无序列表item2
  + 无序列表缩进item通过tab缩进
```
效果如下：
+ 无序列表item1
+ 无序列表item2
  + 无序列表缩进item通过tab缩进

## 代码块
3个反引号```+程序语言(e.g. Java,html)换行后即可输入代码块 ，代码块的每一行缩进至少4个空格或一个制表符，效果如下：
```htmL

    <html>
      <p>我是代码引用示例</p>
    </html>
```

单行代码通过在代码行两侧加上一个反引号`<br>`

## 分割线
使用三个星号
```markdown
我是分割内容
***
```

效果如下：
我是分割内容
***

## 链接
### markdown语法
```markdown
这是一个链接[链接名称](https://www.baidu.com "链接title,可选，鼠标停留展示内容")
```
效果如下：
这是一个链接[链接名称](https://www.baidu.com "链接title,可选，鼠标停留展示内容")

### 地址可点击
如下是e-mail和url地址变成可点击链接
```markdown
<https://www.baidu.com>
<test@163.com>
```
效果如下：
<https://www.baidu.com>
<test@163.com>

### 链接斜体或粗体
```markdown
我是一个 **[加粗链接](https://www.baidu.com)**
我是一个 *[斜体链接](https://www.baidu.com)*
请参考如上 [`代码`](#代码)部分
```
效果如下：
我是一个 **[加粗链接](https://www.baidu.com)**
我是一个 *[斜体链接](https://www.baidu.com)*
请参考如上 [`代码`](#代码)部分
